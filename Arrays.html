Static Arrays

A static array is a container for objects of the same type. It is fixed in size and cannot change the number of elements it contains. Elements are stored next to each other in memory, which makes access very fast (O(1) time).

Example:

int numbers[5]; // array of 5 integers
int numbers[5] = {0, 1, 2, 3, 4} // create an array with elements.

Access

Can access each item:

cout << numbers[3]; // prints element at numbers[3].

Can change items:

numbers[3] = 0; // 4th element is now 0.

Out of Bounds Access

If you try to access an invalid element in an array, it results in undefined behavior.

std::arr<int, 5> = {1, 2, 3, 4, 5}
std::cout << arr[7]; // Undefined Behaviour

What happens is unpredictable. The program might:

  • Crash immediately,

  • Print a garbage value,

  • Or even appear to work as expected.

If you try to overwrite an invalid element, anything can happen. You could crash the program or corrupt data used elsewhere:

std::arr<int, 5> = {1, 2, 3, 4, 5}
arr[7] = 7; // Undefined Behaviour

Array Length

Since arrays contain the same type of elements, their length (how many elements it stores) can be calculated by dividing the total size of the array by the size of a single element:

int arrLength = sizeof(numbers) / sizeof(numbers[0]); // prints number of elements in numbers[].

Inserting an element to an array:

If there’s no unoccupied space in the array, you will have to overwrite data, which means losing data.

First, let’s assume we have enough space

int numbers[10] = {0, 1, 2, 3, 4,}; 
// elements at numbers[5..10] are not assigned
// but they are reserved for potential items.
// You can imagine the array like this:
// numbers[10] = {0, 1, 2, 3, 4, N/A, N/A, N/A, N/A, N/A};
int arrLength = sizeof(numbers) / sizeof(numbers[0]);
int pos = 3; // place where we want to insert item (numbers[3]).
int num = 10;
// shift all the items to get a free position.
for (int i = arrLength; i > pos; i--) {
 numbers[i] = numbers[i + 1]
}

// now numbers[] looks like this:
// numbers[10] = {0, 1, 2, N/A, 3, 4, N/A, N/A, N/A, N/A, N/A}

// insert the element at the position we want
numbers[3] = 10;

// now numbers[] looks like this:
// numbers[10] = {0, 1, 2, 10, 3, 4, N/A, N/A, N/A, N/A, N/A}

What if there is no space for insertion? Then we have to overwrite some data.

  1. We can directly assign value such as:

    numbers[3] = 100;

  2. But what if we want to protect the data at numbers[3]? Then we have two options: we either shift left or right.

    1. Left Shift (overwrite data at [0]):

      int numbers [5] = {0, 1, 2, 3, 4};
int numbersLength = sizeof(numbers) / sizeof(numbers[0]);
int pos = 3; // position where we want to insert element.
int num = 100;

// shift elements to left
for (int i = 0; i < pos; i++) {
 numbers[i] = numbers[i + 1];
}

// now our numbers[] looks like this: numbers[5] = { 1, 2, 3, 3, 4}
// now we can assign our num to numbers[3].

numbers[3] = num;

// now our numbers[] looks like this: numbers[5] = { 1, 2, 100, 3, 4}

    2. Similarly, we can do it for the right shift:

      // right shift
for (int i = numbersLength; i > pos; i--) {
    numbers[i] = numbers[i - 1];
}
// now our numbers[] looks like this: numbers[5] = { 0, 1, 2, 3, 3}
// now we can assign our num to numbers[3].

numbers[3] = num;

// now our numbers[] looks like this: numbers[5] = { 0, 1, 2, 100, 3}
